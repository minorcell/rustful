# 数据类型

## 标量类型

Rust 有四种主要的标量类型：整型、浮点型、布尔型和字符型。

### 整型

整型用于表示整数。Rust 支持有符号和无符号的整型，常见的有 i8、i16、i32、i64、u8、u16、u32、u64、isize、usize 等。

```rust
let x: i32 = 42;
let y: u8 = 255;
```

- 这里我们声明了一个有符号的 32 位整型变量 x，并将其初始化为 42；
- 还声明了一个无符号的 8 位整型变量 y，并将其初始化为 255；
- 默认情况下，整型变量是 i32 类型；

- isize 和 usize 类型根据目标平台的指针大小进行调整，在 32 位系统上为 32 位，在 64 位系统上为 64 位；

### 浮点型

浮点型用于表示带小数点的数字。Rust 支持两种浮点类型：f32 和 f64。

```rust
let a: f32 = 3.14;
let b: f64 = 2.718281828459045;
```

- 这里我们声明了一个 32 位浮点型变量 a，并将其初始化为 3.14；
- 还声明了一个 64 位浮点型变量 b，并将其初始化为2.718281828459045；
- 默认情况下，浮点型变量是 f64 类型；

### 布尔型

布尔型用于表示真（true）和假（false）两种状态。

```rust
let is_active = true;
let is_admin: bool = false;
```

## 字符型

char 类型用于表示单个 Unicode 字符，占用 4 个字节。

```rust
let letter: char = 'A';
let emoji: char = '😊';
```

- 这里我们声明了一个字符型变量 letter，并将其初始化为字符 'A'；
- 还声明了一个字符型变量 emoji，并将其初始化为表情符号 '😊'；
- char 类型可以表示任何 Unicode 字符，包括字母、数字、符号和表情符号；

## 复合类型

Rust 有两种主要的复合类型：元组（tuple）和数组（array）。

### 元组

元组用于将多个不同类型的值组合在一起，并且长度是固定的。

```rust
let person: (i32, f64, char) = (30, 5.9, 'M');
```

- 这里我们声明了一个元组 person，包含一个整型、一个浮点型和一个字符型；
- 可以通过索引访问元组中的元素，例如 person.0 访问第一个元素

### 数组

数组用于存储相同类型的多个值。数组的长度是固定的。

```rust
let numbers: [i32; 5] = [1, 2, 3, 4, 5];
```

- 这里我们声明了一个整型数组 numbers，包含 5 个元素；
- 数组的长度是固定的，不能动态改变；
- 可以通过索引访问数组中的元素，例如 numbers[0] 访问第一个元素
- 数组也可以使用简化的语法来初始化，例如 let numbers = [0; 10]; 创建一个包含 10 个元素且每个元素都为 0 的数组；
- 元组和数组都是在栈上分配内存的，适合存储小型数据集合；
- 对于大型数据集合，建议使用堆分配的数据结构，如 Vec<T>；
