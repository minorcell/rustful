# fn 函数

## 简介

在 Rust 中，函数（function）是组织代码的基本单元。函数通过 `fn` 关键字定义，允许我们将代码封装在可重用的块中。函数可以接受参数并返回值，从而实现代码的模块化和复用。

## 定义函数

函数的基本语法如下：

```rust
fn function_name(parameter1: Type1, parameter2: Type2) -> ReturnType {
    // 函数体
}
```

- `fn`：关键字，用于定义函数。
- `function_name`：函数的名称，遵循标识符命名规则。
- `parameter1: Type1, parameter2: Type2`：函数的参数列表，每个参数由名称和类型组成，多个参数之间用逗号分隔。
- `-> ReturnType`：可选部分，指定函数的返回类型。如果函数不返回值，可以省略此部分。
- 函数体：包含函数执行的代码块。

## 函数的返回值

函数可以通过 `return` 语句返回值，或者直接在函数体的最后一行表达式中返回值。例如：

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b // 直接返回表达式的值
}
```

## 语句和表达式

在 Rust 中，语句（statement）和表达式（expression）是两种不同的代码构造。语句执行某些操作但不返回值，而表达式计算一个值并返回它。

例如：

```rust
let x = 5; // 这是一个语句
let y = {
    let z = 10; // 这是一个语句
    z + 5      // 这是一个表达式，返回值为 15
};
```

在上面的例子中，`let x = 5;` 是一个语句，而 `z + 5` 是一个表达式，它的值被赋给了变量 `y`。

## 注意事项

- 函数参数必须指定类型。
- 函数可以没有参数和返回值。
- 函数体中的最后一个表达式的值会作为函数的返回值，除非使用 `return` 语句显式返回值。
- Rust 使用大括号 `{}` 来定义函数体。
- 函数名称通常使用蛇形命名法（snake_case）。
