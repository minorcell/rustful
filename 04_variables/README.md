# 变量

## variables

在 rust 中，使用 let 关键字来声明变量

```rust
let x = 5;
```

- 这里我们声明了一个名为 x 的变量，并将其初始化为 5；

默认情况下，变量是不可变的。如果你想创建一个可变的变量，需要使用 mut 关键字

```rust
let mut y = 10;
y = 15;
```

- 这里我们声明了一个名为 y 的可变变量，并将其初始化为 10，随后我们将 y 的值修改为 15；

## 常量

在 rust 中，使用 const 关键字来声明常量

```rust
const MAX_POINTS: u32 = 100_000;
```

- 这里我们声明了一个名为 MAX_POINTS 的常量，并将其初始化为 100000；
- 常量的命名通常使用大写字母和下划线分隔单词，以提高可读性；
- 常量必须在声明时初始化，并且其值在程序运行期间不能改变；
- 常量可以指定类型，这在某些情况下是必要的，例如当常量的值不是字面值时；

## 变量遮蔽

在 rust 中，可以通过重新声明一个同名变量来实现变量遮蔽（shadowing）

```rust
let x = 5;
let x = x + 1;
let x = x * 2;
```

- 这里我们首先声明了一个名为 x 的变量，并将其初始化为 5；
- 然后我们重新声明了一个同名的变量 x，并将其初始化为原来的 x 加 1，即 6；
- 最后我们再次重新声明了一个同名的变量 x，并将其初始化为新的 x 乘以 2，即 12；

- 通过变量遮蔽，我们可以在不使用 mut 关键字的情况下修改变量的值，同时保持变量不可变的特性；
- 变量遮蔽还允许我们改变变量的类型，例如：

```rust
let spaces = "   ";
let spaces = spaces.len();
```
